The Head/global Object

Conceptual overview of the Head Object

JavaScript code, itself , must be contained with in aboject. As an example, when crafting JavaScript code for web browser environment, JavaScript is contained and executed within the window object. This window object is concidered to be the "head object", or sometimes confusingly refered to as "the global object"

The Head object is setup by JavaSript behing the scenes to encapsulate user-defined code and to house the native code with which JavaScript comes prepackaged. User-defined code is placed by JavaScript inside the head object for execution.
